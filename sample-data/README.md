# Sample Data

## Datasets

### Taxi Trip Data
- **File**: `sample_taxi_trips.csv`
- **Records**: Sample taxi trip records
- **Purpose**: ETL operations and aggregations

### Housing Market Data
- **File**: `sample_housing_market.csv`
- **Purpose**: ML model training

## Schema

### Taxi Data
- trip_id, trip_date, hour_of_day
- pickup/dropoff coordinates
- trip_distance, fare_amount
- passenger_count, tip_indicator

### Housing Data
- house_id, price (target variable)
- total_rooms, bedrooms, bathrooms
- square_feet, age_years
- distance_to_center, crime_rate, school_rating
